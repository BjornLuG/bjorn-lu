Overview

Visualize Material Design's elevations easily with customizable settings and CSS output. Download the required CSS or SCSS files too.


Project

  • Material Shadow Preview website
  • GitHub repo

Made with Vue, Vuex, Vue Router, Vuetify and Highlight.js.


What I've learned

Working with Vue was a ton of fun, I get to focus on the content and logic rather than on panning the structure. I initially started off without a component UI framework, but as components get more complex, it became time-consuming to reinvent the wheel so I opt for Vuetify instead. The hardest part was to display highlighted code and to enable copy and download of it. Alas, they are solved with Highlight.js and some JavaScript sorcery.
