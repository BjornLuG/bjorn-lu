What I've learned

The project relied heavily on HTML canvas, which I had avoid for a long time because of its complexity. Initially, I opted for HTML2Canvas for the box generation, but since it didn't have a solid support for box shadows, I decided to roll out my own generation system. Turns out it was really simple! I was amazed of how easy it was to convert the data to PNG too.
